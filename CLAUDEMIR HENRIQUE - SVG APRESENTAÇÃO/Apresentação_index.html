    O que é SVG?

    É uma linguagem de marcação para gráficos vetoriais bidimensionais. Ele permite criar e editar gráficos usando tags XML. Os gráficos SVG são escaláveis, o que significa que podem ser redimensionados sem perder qualidade.
    Eles são amplamente utilizados em web design, aplicativos móveis e outros contextos onde a flexibilidade e a qualidade são importantes.
