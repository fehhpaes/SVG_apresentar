O que são?

SVG significa Scalble Vetor Graphics, uma imagem vetorial formato XML Além disso, como um arquivo vetorial, ele pode ser aumentado ou reduzido sem perder resolução

XML é uma linguagem mais s Aplicar efeitos e filtro dentro da imgem diretamente através do CSS. ótimo suporte para navegadores, até o IE8 Boa acessibilidade para leitores de telas Possui árvore no DOM, pode ser inspecionado através do browser icones patterns imagens vetores
